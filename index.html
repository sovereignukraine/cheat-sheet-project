Semantic HTML elements

Description of semantic elements

Element Description Atributes
<header> Usually used for either navigational links or introductory content containing <h1> to <h6> headings. "class" or "id" atribut
<nav> Used to define a block of navigation links such as menus and tables of contents. "class" or "id" atribut
Element Description Atributes
<main> Used to encapsulate the dominant content within a webpage.
<article> Holds content that makes sense on its own. <article> can hold content such as articles, blogs, comments, magazines, etc.
<section> Defines elements in a document, such as chapters, headings, or any other area of the document with the same theme.
<aside> Used to mark additional information that can enhance another element but isnâ€™t required in order to understand the main content.
<figure> Used to encapsulate media such as an image, illustration, diagram, code snippet, etc, which is referenced in the main flow of the document.
<figcaption> Used to describe the media in the <figure> tag. Usually, <figcaption> will go inside <figure>.
<audio> Used to embed audio content into a document. "src" - specifies the URL of the audio file
"controls" - dispaly audio controls.
<video> Using to add video to the website. "controls" - dispaly video controls.
"autoplay" - automatically playing as soon as the page is loaded.
"loop" - video continuously playing on repeat.
"src" - specifies the URL of the video file.
<embed> Can embed any media content including videos, audio files, and gifs from an external source. "src" - specifies the URL of the video file.
Element Description Atributes
<footer> The footer contains information such as: Contact information, Copyright information, Terms of use, Site Map, Reference to top of page links. "class" or "id" atribut
